# 🚢 Weldedron - Cold Chain Logistics Website

A modern, responsive landing page designed for **Weldedron**, a cold chain logistics and transportation company. The website showcases the company's logistics services, reefer transportation, freight forwarding, supply chain solutions, and associated transport network with a clean and professional user interface.

---

## 📌 Project Overview

This project is built as a responsive company website using **HTML**, **Tailwind CSS**, and **Font Awesome**. It focuses on delivering a modern user experience while maintaining fast loading performance and mobile responsiveness.

---

## ✨ Features

- Responsive Hero Section
- Modern Navigation Bar
- Company Introduction
- Cold Chain Logistics Services
- Service Cards with Hover Effects
- Associated Company (Dealex) Section
- Call-to-Action Section
- Professional Footer
- Mobile-Friendly Design
- Clean & Organized Code Structure
